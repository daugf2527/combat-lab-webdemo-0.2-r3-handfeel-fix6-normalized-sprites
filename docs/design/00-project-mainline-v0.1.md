## 1. 一句话主线

这是一个以类 DNF 的 2.5D 战斗打怪与角色养成为表层载体，以 AI 时代“外智代役、借智有债、代役留责”为暗线核心的游戏化智能体养成系统。

玩家在表层看到的是战斗、刷图、技能、装备、成长和副本推进；在深层，现实中的 AI 工具调用、插件任务、MCP、Skill、Claude Code、Codex、VS Code 等外部能力，会被抽象为寄宿在人身上的“外智之力”。人把原本需要自己思考和执行的事情交给外物，外物代人行事，任务完成后的成果回流到游戏角色，化为经验、道具、技能、货币和副本进度。

但这股力量不是白给的。借智有债，代役留责。AI 可以替人思考、执行、提交产物，但审核、兜底、责任和后果仍会回到人身上。

---

## 2. 表层阳线：积极、明亮、可玩的战斗成长世界

游戏表面必须是阳光、积极、热血、成长向的。玩家第一眼看到的不是 AI 焦虑，而是一个可玩的 2.5D 战斗世界。

表层关键词：

- 类 DNF 的 2.5D 横版 / 斜视角战斗
- 主角、小怪、Boss、技能、连招、受击、击退、硬直
- 副本推进、装备掉落、角色成长、技能强化
- 玩家可以手动操作，也可以通过后续系统让外部任务成果反哺角色
- 表层体验强调爽感、成长感、探索感、挑战感和成就感

阳面的核心价值：

1. **增能**：普通人借助外智获得过去只有团队、高手或公司资源才能提供的能力。
2. **成长**：现实任务成果可以转化为角色成长，劳动反馈被游戏化、可视化。
3. **创造**：玩家不只是打怪，也是在通过真实任务建设自己的角色世界。
4. **越级挑战**：外智让低等级角色有机会挑战过去不可触及的高级任务。
5. **共生**：理想状态不是人被 AI 替代，而是人学会调度、约束、验收和驾驭外智。

表层叙事应尽量明亮，不直接使用过度阴暗的词汇作为门面。暗线可以藏在系统、剧情和中后期展开中。

## 3. 深层暗线：AI 时代的人与外智

暗线讲的是 AI 时代的真实处境。

人原本需要亲自劳心劳力完成一件事，但现在把“思考过程”和“执行过程”交给了外物。这个外物不是普通工具，而像被请来的灵、被封入体内的外来力量、可借用的未来能力。

它初期只是助手，替人做 5 到 10 分钟的执行；之后能规划、评估、拆解任务，替人完成 10 到 30 分钟的工作；再往后进入全量 Agent 阶段，能接管 8 到 10 小时级别的人类工作流。最终，人可能只剩审核、签字、授权和背锅。

暗线关键词：

- 外智代役
- 借智有债
- 代役留责
- 智力平权后的智力贬值
- 成长阶梯断裂
- 新人无怪可打，老人无价可卖
- 硅基繁荣，碳基滞涨
- 人以为在控制 AI，AI 却在重塑人类社会结构

暗线不是简单反 AI，而是承认 AI 的增能价值，同时揭示其代价：依赖、债务、反噬、责任转移、阶层重塑、劳动贬值和人的自我退场。

## 4. 个体层主线

个体层的矛盾是：

> 我能不能借外智之力，却不把自己抵押给外智？

主角不是传统救世主，而是一个处在 AI 时代断代现场的人。

他知道不用外智就会落后；也知道全靠外智会空心。他想借光变强，又不想被光夺目。他想越级挑战高级副本，又不想失去真正打怪升级的本能。

个体层核心循环：

```text
现实任务
  ↓
请外智代役
  ↓
AI 调用工具 / 插件 / MCP / Skill / 开发环境
  ↓
任务完成
  ↓
成果转化为游戏收益
  ↓
角色成长
  ↓
产生代价、债务、责任或反噬风险
  ↓
玩家通过审核、手动战斗、复盘、修正来维持本心
```

## 5. 社会层主线

放大到社会，游戏世界处于“智力平权后的失价时代”。

AI 让普通人获得过去很难拥有的能力，也让传统高智力劳动的议价权下降。资本不再只为人的智力高额付费，而是转向投资模型、算力、Agent、自动化平台和 AI 周边生态。

传统成长路径被打断：

```text
初级任务 → 中级项目 → 复杂系统 → 踩坑复盘 → 高级判断
```

外智出现后，很多人跳过中间磨砺，直接借 AI 挑战高级任务。表面是越级成长，实际可能是登阶断裂。

社会层口号：

- 新人无怪可打，老人无价可卖。
- 越级者满城，登阶者绝迹。
- 硅基灯火日盛，碳基薪火渐寒。
- 人以为自己在役灵，灵却借人间之手供养自身。

## 6. 人群谱系草案

这些不是固定职业，而是人在 AI 时代与外智之间的关系类型。

| 类型 | 表层身份 | 深层含义 |
|---|---|---|
| 掌灯人 / 操机员 | 会调度外智的人 | 能下令、拆任务、验收、修正，但仍需守住人的判断 |
| 司灯者 / 资源掌控者 | 掌握外智入口的人 | 控制模型、算力、平台、任务分发和收益结算 |
| 炼灯师 / 工具构造者 | 训练和封装外智的人 | 写 Skill、MCP、插件、提示词和自动化流程 |
| 验灯官 / 审核者 | 审核外智产物的人 | 活不是他干的，但错要他认 |
| 灯奴 / 被流程支配者 | 被外智任务流统治的人 | 看似效率高，实际自主值下降 |
| 无灯人 / 边缘者 | 无法或拒绝接入外智的人 | 成长慢，但保留完整的自我和判断 |
| 契债商 / 平台商 | 卖额度、卖算力、卖工具的人 | 靠别人的灵债与依赖获利 |
| 灰灯客 / 灰产者 | 钻规则、偷收益的人 | 利用漏洞、伪造结果、刷奖励 |
| 斩灯人 / 反抗者 | 反对或猎杀失控外智的人 | 他们反对的不是技术，而是人对外物的跪拜 |
| 空灯者 / 失能者 | 失去外智后也失去自身能力的人 | 长期依赖后，一旦灯灭，自己也不会发光 |

后续可以把这些类型转化为 NPC、阵营、敌人、任务线、剧情路线或玩家发展方向。

## 7. 暂定核心术语

> 注意：以下术语均为暂定。后续正式命名应保持表面积极、明亮、克制，避免过早暴露暗线。

| 术语 | 暂定含义 |
|---|---|
| 外智 | AI 在世界观中的抽象，不直接叫 AI |
| 代役 | 外智替人完成现实任务或工具流程 |
| 返炁 / 归炁 | 任务成果回流为角色成长资源 |
| 灵债 / 回响 | 使用外智产生的成本、依赖、错误与责任 |
| 本心 / 自明 | 人自己的判断力、控制力和真实成长 |
| 晦影 | 外智过度使用后的暗面风险 |
| 明庭 / 灵台 | Web 平台、任务大厅、收益中枢的世界观称呼 |

## 8. 当前阶段原则

1. **先把游戏做成**：战斗手感、怪物反馈、Boss 表现、技能释放、素材稳定优先。
2. **再把 AI 做成力量系统**：外智不是按钮，而是角色成长与任务收益系统的内核。
3. **再接真实任务**：优先从 GitHub、CI、PR、代码修复、文档生成等开发任务开始。
4. **最后做平台生态**：Web 平台、插件、MCP、Skill、VS Code、Claude Code、Codex、App 打包和奖励体系逐步接入。

---

## 9. 当前最重要的一句话

> 这是一个类 DNF 的 2.5D 战斗养成游戏。人在现实中借外智代役，外智完成事务后返还力量；力量进入游戏角色，化为战斗、成长、装备与副本进度。但借来的光终要偿还，人若失去本心，就会被外智反过来塑形、牵制，甚至掏空。
