# Berserker 核心动作逐帧校准表
> **Status: [BERSERKER] — 狂战士核心动作逐帧校准表**

- **嗜血 (Bloodlust)**：按 →→+Z 施放，角色向前伸手抓取范围内一名敌人并吸收其能量后向前爆发，对抓点和后方敌人造成伤害【67†L33-L42】。若目标为不可抓取（建筑、Boss、霸体怪等），角色会撕取目标血液并向前喷发，对周围目标造成伤害【67†L41-L44】；若抓取失败，依然触发怒爆效果生成血能爆炸【67†L47-L50】。嗜血基础冷却6秒，可受狂暴（Frenzy）效果降低【67†L60-L68】。
- **暴走狂怒 (Raging Fury)**：按 ↓↑+Z 施放，角色向上掀起周围敌人并造成伤害，同时前方放出冲击波，接着在地面生成大量血柱连续攻击敌人【70†L35-L40】。血柱攻击次数多（约10次），冲击波再加1次，总计约11次伤害。命中后敌人浮空后落地【70†L35-L40】。
- **快速反冲 (Quick Rebound)**：击倒起身时自动触发，角色蹲下进入无敌状态，最长可保持2秒后转为短暂霸体【65†L190-L197】。按住不松则持续无敌，松开或时限到后立即恢复可行动。
- **普攻取消**：狂战大部分技能（包括嗜血、暴走狂怒等）均可被基本攻击取消（即施放后可用普攻打断后摇）【67†L60-L63】【70†L46-L49】。
- **狂暴 (Frenzy) 效果**：狂暴现为对所有非觉醒技能提供冷却缩减（每级10%），取代了之前仅针对个别技能的减CD效果【85†L352-L359】。也对狂战多个技能附加流血效果并增加伤害。
