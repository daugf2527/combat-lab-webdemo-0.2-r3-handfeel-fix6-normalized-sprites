# Amplification (增幅) — DFO World Wiki Reference

> Source: https://wiki.dfo.world/view/Amplification
> Scraped: 2026-05-01

---

Amplification is used for gear enhancement, similar to Reinforcement and Refinement. It adds Interdimensional Ability stats (STR/INT/VIT/SPI) in addition to standard Reinforcement bonuses.

## How to Obtain Interdimensional Gear

### Option 1: Contaminated Equipment

- Level 55+ Epic/Mythic equipment can drop with "tainted by Otherverse Energy"
- Must purify at Klonter (Aphelia Post) using an Otherverse Energy Purification Scroll
- After purification, gear gains a random Interdimensional Ability (STR/INT/VIT/SPI)
- Use Conversion Spell to re-roll the ability type
- Purified gear can only be upgraded through Klonter or Kiri

### Option 2: Endowment (Manual)

- Use Pure Tome, Silver Amplification Grimoire, or Golden Amplification Grimoire
- Grimoires often obtained from events
- Requires Lv 55-110 Rare+ equipment

### Option 3: Safe Amplification

- Season 7+ safeguard system
- Failure accumulates 5-10% adjustment values (resets on success)
- Requires Lv 100-105 Unique/Legendary/Epic/Mythic with +9 or lower
- Uses Harmonious Crystals or Radiant Harmonious Crystals (not Crystallized Chaos)
- Safe Amplify only goes up to +10

## Amplification Failure Penalty

| Upgrade | Can Fail | Failure Penalty |
|---------|----------|-----------------|
| +0 → +4 | No | None |
| +4 → +5 | Yes | Loses one upgrade level |
| +5 → +6 | Yes | Loses one upgrade level |
| +6 → +7 | Yes | Loses one upgrade level |
| +7 → +8 | Yes | Loses three upgrade levels |
| +8 → +9 | Yes | Loses three upgrade levels |
| +9 → +10 | Yes | Loses three upgrade levels |
| +10 → +11+ | Yes | Item breaks |
