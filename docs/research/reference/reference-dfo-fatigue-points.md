# Fatigue Points (疲劳值) — DFO World Wiki Reference

> Source: https://wiki.dfo.world/view/Fatigue_Points
> Scraped: 2026-05-01

---

Fatigue is a system that directly affects character gameplay for exploring content. Base: 156 FP per weekday, 176 FP on weekends.

## FP Consumption Rules

- Entering a dungeon consumes 1 FP.
- Each room progressed adds 1 FP consumed.
- Returning to a previously visited room does NOT consume FP.
- Having 0 FP mid-dungeon does not prevent proceeding — run full maps when approaching 0.
- Special Dungeons cost 8 FP for entry but no additional FP per room.

## FP Recovery

- Fully replenished at 9:00 AM UTC (daily reset).
- Can be artificially replenished via Alchemy potions or Mileage shop items.
- Unused FP contributes to Growing Fatigue Points the next day, up to 117 max (higher cap for higher-level characters). Grows double EXP while they last.

If both contracts are active, Plus benefits apply first.

## FP Express Mode

Under NeoPremium Plus, FP Express mode doubles FP consumption in dungeons but increases EXP gain.

## Burning Event

Rare event (typically Fri-Sun) providing tiered bonuses based on account Fatigue:
- Bonus EXP/Drop rates
- Skill level increases
- Cooldown reductions
- Increased FP cap during event
