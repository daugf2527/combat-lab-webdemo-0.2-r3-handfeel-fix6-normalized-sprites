# Avatar / Costume System (装扮系统) — DFO World Wiki Reference

> Source: https://wiki.dfo.world (Avatar, Emblem, Bind Cube pages)
> Scraped: 2026-05-02

---

The avatar system is DNF's core monetization and character customization layer. Avatars provide combat stats, set bonuses, and cosmetic appearance. The system spans 11 slots, 5 rarity tiers, emblem socketing, synthesis, and a dedicated trading market.

---

## Avatar Slot System (11 Slots)

| # | Slot | Primary Stat Options | Emblem Socket Color |
|---|------|---------------------|---------------------|
| 1 | **Hair** | INT, SPR, Casting Speed, HP/MP Recovery | Red |
| 2 | **Hat** | INT, SPR, Casting Speed, HP/MP Recovery | Red |
| 3 | **Face** | Magic Defense, Hit Recovery, Attack Speed, Abnormal Status Tolerance | Yellow |
| 4 | **Torso** | Magic Defense, Hit Recovery, Attack Speed, Abnormal Status Tolerance | Yellow |
| 5 | **Top** | Skill level +1 (starter or sub-class skills) | Green + Platinum (Rare only) |
| 6 | **Bottom** | Physical Defense, Max HP, Max MP | Green + Platinum (Rare only) |
| 7 | **Waist** | Elemental Resistances, Evasion Rate, Max Weight | Blue |
| 8 | **Shoes** | STR, VIT, Movement Speed, Jump Strength | Blue |
| 9 | **Skin** | Physical/Magic Damage Reduction, Hit Recovery | Multicolor |
| 10 | **Weapon** | Various (Speed, stats selectable) | Multicolor |
| 11 | **Aura** | Elemental Damage, PMI, All Stats | Multicolor |

**Set counting rules**: Only slots 1-8 count toward set bonuses. Skin, Weapon, and Aura do NOT count.

## Avatar Synthesis (Bind Cube System)

### Bind Cube Types

| Type | Cost | Success Rate | Result |
|------|------|-------------|--------|
| **Standard Bind Cube** | ~150-200 CERA | Low (unspecified) | Random Rare Avatar piece |
| **Anniversary Bind Cube** | 150 CERA | Higher than standard | Classic Rare Avatars, higher 1st gen rates |
| **Absolute Bind Cube** | 790 CERA (package) | **100% guaranteed** | One 3rd Gen Rare Avatar of choice |
| **Platinum Rare Clone Weapon Bind Cube** | — | 100% guaranteed | Rare Clone Weapon Avatar |

### Absolute Bind Cube Requirements

- Consumes 8 non-Rare Avatars (one of each main slot: Hair through Shoes)
- Result: One 3rd Generation Rare Avatar piece of player's choice
- Created Rare Avatar is **untradable**
- Cannot use on Skin, Aura, Clone Weapon Avatars, or Combine/Sunder restricted avatars

### Rare Avatar Generations

All generations provide identical stats — only visual design differs. Successive generations (1st through 11th+) represent different visual iterations released over time.

## Emblem System

### Socket Colors and Stats

| Color | Slots | Emblem Stats | Gold Values |
|-------|-------|-------------|-------------|
| **Red** | Hair, Hat | STR, INT, VIT, SPR, All Stats | Single +25, All +9 |
| **Yellow** | Face, Torso | AtkSpeed, CastSpeed, HP/MP Recovery, Hit Recovery | AtkSpeed +1.5%, CastSpeed +2.2% |
| **Green** | Top, Bottom | Phys/Mag Crit, Max HP/MP, Evasion | Crit +3%, HP/MP +160 |
| **Blue** | Waist, Shoes | Hit Rate, MoveSpeed, Phys/Mag Defense, Jump | HitRate/MoveSpeed +1.5%, Def +620 |
| **Multicolor** | Skin, Weapon, Aura | Max Weight, All Elem Resist, All Abn Status Resist | Weight +3.5kg, Resistances +5 |
| **Platinum** | Top, Bottom (Rare only) | Skill level increases + All Stats +6 | Legendary grade |

### Emblem Grades

| Grade | Rarity | Acquisition |
|-------|--------|-------------|
| **Bronze** | Uncommon | Dungeon drops, lowest synthesis |
| **Silver** | Rare | Avatar Sundering, synthesis from Bronze |
| **Gold** | Unique | Avatar Sundering, synthesis from Silver |
| **Platinum** | Legendary / Uncommon | Avatar Sundering (Rare Avatars), dungeon drops, Auction House |

### Emblem Rules

- Once equipped, emblem **cannot be removed** — replacing destroys the old emblem
- Solid emblems must match socket color; Dual emblems fit either of their two colors
- Multicolor emblems go anywhere except Platinum sockets
- Platinum emblems become **account-bound after one trade**
- Sundering avatars at NPC Daphne produces random emblems; better results if avatar has Gold emblems equipped

### Platinum Emblem Details

- Increase specific class basic or advancement skill levels
- Provide +6 to all stats (STR, INT, VIT, SPR)
- **Uncommon Platinum**: Boosts level 1 skills (e.g., Upward Slash +1)
- **Legendary Platinum**: Boosts mid-to-high level advancement skills — most valuable emblems
- 2 Legendary Platinums can be rerolled into 1 random Legendary Platinum

## Avatar Trading and Binding

### Tradable Avatars

- CERA shop Look Avatars: tradable until equipped or dyed
- Bind Cube Rare Avatars: tradable until equipped, combined, or sundered
- Some package avatars: check package description

### Untradable Avatars

- Absolute Bind Cube results: always untradable
- Clone Avatars from packages: typically untradable
- Event avatars: usually untradable
- Any avatar becomes untradable after: equipping, dyeing, combining, or sundering

### Avatar Market

- Dedicated trading channel for avatar bundles
- Functions: search, purchase, sell bundles, browse all released sets
- Hall of Fashion: most-liked avatar combinations per season
- Showcase Avatar: share designs, like others' designs

## Special Avatar Types

### Weapon Avatar

- Optional cosmetic changing weapon appearance
- Clone Weapon Avatar: selectable stats (+45), has emblem sockets
- Rare Clone Weapon Avatar: higher stats, via Platinum Rare Clone Weapon Bind Cube
- Does NOT count toward set bonuses

### Aura Avatar

- Visual effect around character
- Example stats: +20 Elemental Damage, +50 PMI, +100 All Stats (varies by package)
- Multicolor emblem sockets (some have 2)
- Does NOT count toward set bonuses

### Skin Avatar

- Defensive properties: Physical or Magical Damage Reduction +3-3.5%, Hit Recovery +80-90
- 2 Multicolor emblem sockets
- Does NOT count toward set bonuses
